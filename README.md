# MoreFind
一款用于快速导出URL、Domain和IP的小工具


## 快速安装
方式一: 通过Go包管理安装
```bash
go get github.com/mstxq17/MoreFind
```
方式二: 直接安装二进制文件

方式三: 本地编译


## 用法说明
1)帮助信息
```bash
MoreFind -h
```
![img.png](img/img.png)

2)导出URL
```bash
MoreFind -u
```

3)导出域名
```bash
MoreFind -d
```

4)导出ip
```bash
MoreFind -i
```
